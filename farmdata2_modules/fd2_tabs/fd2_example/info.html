FarmData2 Example Module

A sample module for learning about and experimenting with FarmData2 modules.

This module provides examples of how to create and use common UI elements, make API requests and also how to perform testing. It illustrates a number of the common idioms that are used in FarmData2 modules. It also serves as a template for other modules such as the FieldKit (fd2_field_kit) and the BarnKit (fd2_barn_kit).
